โปรเจกต์นี้เป็นแบบฝึกหัด Midterm  
วิชา Web Development

## Features

- บวก ลบ คูณ หาร (+, -, ×, ÷)
- คำนวณอัตโนมัติเมื่อกด operator
- ป้องกันการหารด้วยศูนย์ (Division by Zero)
- ปุ่ม Clear (AC)
- แสดงประวัติการคำนวณ (History) โดยรายการล่าสุดแสดงด้านบน
